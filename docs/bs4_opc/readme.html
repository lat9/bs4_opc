What it does

This sub-template builds on the ZCA Bootstrap 4 (v3.1.0 or later), providing Bootstrap styling for the Zen Cart One-Page Checkout plugin (v2.3.7 or later).

Installation

This plugin provides a Bootstrap 4 rendering for the One-Page Checkout pages. You should have previously installed:

  1. The 'base' ZCA Bootstrap 4 template, v3.1.0 or later: https://www.zen-cart.com/downloads.php?do=file&id=2191
  2. The 'base' One-Page Checkout plugin, v2.3.7 or later: https://www.zen-cart.com/downloads.php?do=file&id=2095

Once those 'base' file-sets have been installed, you're ready to install the 'sub-template' files.

  1. Note: Don't copy OPC's /includes/templates/YOUR_TEMPLATE/css/login.css file to your bootstrap template's sub-directory or to any bootstrap-clone that you've made. If that file has already been copied, be sure to remove it.
  2. Unzip the template's package file, which you've most likely already done if you're reading this!
  3. Copy all the files present to your storefront, overwriting the 'base' bootstrap template's directories.
  4. If you've made a clone of the bootstrap template
    • Rename the bootstrap sub-directories in this sub-template's distribution to match the name of your "clone".
    • Copy (or merge) all the files to your clone also.
  5. Delete the no-longer-used /includes/templates/bootstrap/css/login.css that is distributed by One-Page Checkout from both the 'base' bootstrap template and your clone.
